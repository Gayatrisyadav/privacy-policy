Privacy Policy for Fake Profile Detection Project Introduction: This Privacy Policy explains how we collect, use, and share data from Instagram users for the Fake Profile Detection project. Data Collection: We access Instagram's public API to collect publicly available profile data for our project. This data may include usernames, profile pictures, bio text, and public posts. The data will not be shared or used for any other purposes. Data Retention: We will retain collected data for the duration of the project. After the project is completed, all data will be deleted.
